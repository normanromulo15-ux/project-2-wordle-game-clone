# PSEUDOCODE

1\. SELECT ONE RANDOM 5-LETTER WORD FROM THE ARRAY.

2\. ACCEPT USER INPUT

3\. IF USER INPUT INCLUDES A VALID WORD FROM THE ARRAY, THE PROGRAM CHECKS THE LETTERS BY ORDER. ELSE, IT SHOWS AN ERROR MESSAGE ON THE UI SAYING INVALID WORD.

4\. IF A LETTER IS PLACED AT THE EXACT ORDER, IT WILL BE HIGHLIGHTED AS GREEN

5\. IF A LETTER IS PLACED WITHIN THE WORD BUT NOT AT THE EXACT ORDER, IT WILL BE HIGHLIGHTED AS YELLOW

6\. IF A LETTER IS INCLUDED IN THE WORD, IT WON'T BE HIGHLIGHTED

7\. STEPS 3 - 6 CONTINUES FOR 6 ATTEMPTS

8\. IF THE USER DOES NOT GUESS THE WORD AFTER THE 6TH ATTEMPT, THE GAME IS OVER.

9\. GAME IS OVER IF THE USER GUESSES THE WORD WITHIN 6 TRIES

10\. GAME CAN BE RESTARTED

## REACT STATES TO BE USED

- randomWord
- userInput
- numberOfAttempt
